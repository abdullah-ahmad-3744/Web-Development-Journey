# Web Dev Journey

Welcome to my **Web Development Journey Repository!**  
This repository is a comprehensive record of my growth and exploration as a web developer — covering everything from foundational web technologies to modern full-stack frameworks.

---

## Overview

This repository contains code snippets, projects, and practice exercises as I learn and master various web development technologies, including:

- **HTML** – The structure and backbone of every webpage.  
- **CSS / Tailwind CSS** – Styling and responsive design with both traditional and utility-first approaches.  
- **JavaScript** – Adding interactivity, logic, and dynamic behavior to web applications.  
- **React.js** – Building fast, reusable, and interactive front-end user interfaces.  
- **Node.js & Express.js** – Creating robust back-end APIs and server-side logic.  
- **MongoDB** – Handling and managing data with a flexible NoSQL database.

---

## Purpose

The main goal of this repository is to document my learning process, experiments, and personal projects as I develop a deeper understanding of both front-end and back-end web technologies.  
It serves as both a **learning archive** and a **showcase of progress** toward becoming a proficient full-stack developer.
